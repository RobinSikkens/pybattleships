Ship¶

This module defines the class Ship, and a helper class, ShotResult.

class pybattleships.ship.Ship(x: int, y: int, horizontal: bool, size: int)[source]¶

A Ship represents a Battleship piece.

Parameters:
  • x (int) – The x-coordinate of the top-left square.
  • y (int) – The y-coordinate of the top-left square.
  • horizontal (bool) – Whether the ship is positioned horizontally (True), or vertically (False).
  • size (int) – The number of squares the Ship occupies.
Raises:
  • ValueError – if the Ship is placed outside of the Board.
  • ValueError – if the Ship has an invalid size.
fields¶

Return the list of fields this Ship occupies as a list of (x, y) tuples.

Assert that the given shot was on this Ship, if so, increase the hitcounter. Return whether the Ship was missed, hit, or sunk.

size¶

Return the number of fields the Ship occupies.

Type:int
sunk¶

A ship sinks when every position is hit, which means it has been hit size times. Once the ship has sunk, this property is True.

Type:bool
class pybattleships.ship.ShotResult[source]¶

A ShotResult represents the result of firing on a Ship.

HIT = 1¶

Shot hit Ship, did not sink yet.

LOSS = 3¶

Shot hit Ship, Ship sunk, Ship was last Ship on Board.

MISS = 0¶

Shot did not hit Ship.

SUNK = 2¶

Shot hit Ship, Ship sunk.

Return the character to use when pretty-printing.
