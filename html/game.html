Game¶

This module defines the class Game.

class pybattleships.game.Game(player1: str, player2: str)[source]¶

A Game represents a game of Battleships between two players.

Parameters:
  • player1 (str) – the identifier to use for player one.
  • player2 (str) – the identifier to use for player two.

process_fire(x: int, y: int) → pybattleships.ship.ShotResult[source]¶

Process a player’s input: fire at the opponent’s Board, and return the result of the shot.

Returns ShotResult.LOSS if this shot makes the opponent lose the game.

Parameters:
  • x (int) – the x-coordinate to fire at.
  • y (int) – the y-coordinate to fire at.
setup_board(player_id: str, ships: [<class 'pybattleships.ship.Ship'>]) → bool[source]¶

Create a Board from the given set of ships, assert validity and register the Board with the Game if it is valid.

Returns False if the Board is invalid, and will then not accept the Board.

Parameters:
  • player_id (str) – the player identifier to set the board for.
  • ships ([Ship]) – the list of Ships to populate the Board with.
Raises:

AssertionError – if the passed player identifier is unknown.
start_game() → bool[source]¶

Asserts both boards are present and valid, starts the Game, and gives the turn to player 1.

Returns False if one or more of the Boards is absent or invalid, True on success.
